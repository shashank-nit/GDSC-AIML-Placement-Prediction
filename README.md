## Placement Prediction using Machine Learning

### Objective
To predict whether a student will be placed based on academic performance, skills, and training-related features.

### Dataset
The dataset contains student information such as CGPA, internships, projects, aptitude score, soft skills, and placement status.

### Methodology
- Performed Exploratory Data Analysis to identify key factors influencing placement.
- Encoded categorical variables using Label Encoding.
- Trained a Logistic Regression model for binary classification.
- Evaluated performance using accuracy and classification metrics.

### Key Insights
- Higher CGPA improves placement probability.
- Internships and placement training significantly increase placement chances.
- Aptitude test scores play an important role.

### Result
The model achieved an accuracy greater than 60%, satisfying the induction task requirements.
